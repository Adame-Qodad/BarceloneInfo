Paramètres GitHub

Pour que les modifications soient visibles en ligne sur GitHub Pages, connectez votre dépôt GitHub. Un token avec permission Contents: Read and write est nécessaire.

Comment ça marche ?

  1. Modifiez le contenu dans les onglets Effectif, Actualités ou Club.
  2. Cliquez sur Publier en ligne — les fichiers JSON sont commités sur GitHub.
  3. GitHub Pages redéploie automatiquement le site (1-2 minutes).

Lien d'accès admin : admin.html?key=VOTRE_CLE
Changez la clé dans js/admin/config.js avant la mise en ligne.
